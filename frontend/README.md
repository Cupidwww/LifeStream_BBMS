# frontend

### View
1. 登录（用户名+密码）
2. 注册
3. 捐献者主界面
    - 个人信息（姓名等信息 献血记录 健康信息）姓名等个人信息可以修改
    - 献血预约
4. 工作人员主界面
    - 个人信息 （姓名，工号等信息 献血记录 健康信息）姓名等个人信息可以修改
    - 入库出库（入库检查血液是否合格，出库检查血液库存和血液状态）
    - 血液查询（增删改查）（过期提醒）
5. 管理员主界面
    - 用户管理 用户表
    - 捐献历史表
    - 健康信息表
    - 血液管理表
    - 所有表（增删改查）
